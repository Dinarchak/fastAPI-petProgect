# fastAPI-petProgect
Учебный проект по fastAPI(сервис расписания занятий для студентов)

### Используемые библиотеки
**alembic==1.16.4**

**fastapi==0.116.1**

**SQLAlchemy==2.0.43**

**pydantic==2.11.7**

**typing==3.7.4.3**

И все им необходимое

### Запуск
 uvicorn app.main:app

### Краткое описание
 - Реализованы CRUD опериции для нескольких сущностей(Users, Majors, Students, Events)
 - Есть система авторизации, использующая JWT
 - Работает ролевая система по выдаче прав доступа(user, admin)
